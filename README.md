# Authentication and Authorization Microservice

This is the first implementation of a post series written on my blog. You can see the explanation [here](https://github.com/carlosvilladev/authentication-authorization-microservice)

#### Authentication & Authorization 

Install the dependencies and start the server.

```sh

$ npm install
$ npm start
```
